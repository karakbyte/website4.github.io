Project 1: Smart IoT-Enabled Floral Waste Recycling System for Sustainable Biomaterial Production

Floral waste, consisting primarily of discarded flowers and leaves, is a significant contributor to environmental pollution in many regions Religious rituals, cultural practices, and public gardening activities generate vast quantities of floral waste that is often discarded in landfills or water bodies, leading to water pollution and soil degradation Our project's core innovation lies in the development of a sophisticated IoT-driven system capable of automating the collection, drying, and recycling of floral waste and make it good for the environment

SOFTWARE IMPLEMENTATION

Arduino Firmware: Controls hardware components like ultrasonic sensor, LEDs, servo motor, and LCD.

Mobile App: Provides user interface for monitoring, notifications, and system status via Bluetooth or Wi-Fi.

Cloud Integration: Stores and analyzes data on platforms like ThingSpeak, Firebase, or AWS IoT.

Database Management: Stores system configuration, user profiles, and historical data securely.

System Integration: Ensures seamless communication between software and hardware components.


HARDWARE IMPLEMENTATION

Ultrasonic Sensor: Detects the presence of waste material in the container.

Heater: Provides heat to dry the waste material.

Servo Motor: Controls the opening and closing of the container floor.

LCD Display: Shows system status, countdown timer, and alerts.

Arduino Board: Acts as the central control unit for managing hardware components.

Relay Module: allows Arduinos to control devices that require a higher voltage or current than what the microcontroller can provide.

GPS Module: communicate with the Arduino through serial communication
